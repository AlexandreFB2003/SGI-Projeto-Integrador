Separação de responsabilidades 

Função:

Rotas: Receber dados do front-end e enviar para o controller
Controller: Validar dados | Enviar dados para o Service
Services: Fazer Regras de Negócio (Solicitar o Repository caso necessário)
Models: Modelar Dados (característica do que vai ser modelado) | Modelar BD
Repositories: Manipular Banco de Dados | Ele NÃO processa dados, apenas solicita ao BD


FLUXO DE UMA API:

-> ROTAS -> CONTROLLER -> SERVICE -> REPOSITORY
